
React + Vite + Tailwind CSS Setup (D' Plush Choice)

Step-by-step process:

1. **Create React Vite Project:**
	- Ran: `npm create vite@latest frontend -- --template react`
	- Chose the `react` template to scaffold a modern React app in the `frontend` folder.

2. **Navigate to Project Folder:**
	- Used: `cd frontend`

3. **Install Project Dependencies:**
	- Ran: `npm install` to install all dependencies from `package.json`.

4. **Install Tailwind CSS and Related Packages:**
	- Ran: `npm install -D tailwindcss postcss autoprefixer`
	- Optionally installed: `npm install @tailwindcss/vite` for Vite integration.

5. **Initialize Tailwind CSS Config:**
	- Ran: `npx tailwindcss init -p` to create `tailwind.config.js` and `postcss.config.js`.

6. **Configure Tailwind in Project:**
	- Edited `tailwind.config.js` to set content paths (e.g., `./src/**/*.{js,jsx,ts,tsx}`).
	- Added Tailwind directives to `src/index.css`:
	  ```css
	  @tailwind base;
	  @tailwind components;
	  @tailwind utilities;
	  ```

7. **Clean Up Boilerplate:**
	- Removed default Vite/React template code from `App.jsx`, `index.css`, and other files.
	- Deleted unused assets and example components.
	- Customized the app structure and styles for D' Plush Choice branding.

8. **Run Development Server:**
	- Used: `npm run dev` to start the Vite development server at `http://localhost:5173/`.

9. **Set Up Branding and Favicon:**
	- Added a custom SVG icon to `public/dplushchoice-icon.svg`.
	- Updated `index.html` to use the new icon.

10. **Verify Setup:**
	 - Confirmed Tailwind CSS is working by using utility classes in components.
	 - Ensured the app is clean, fast, and ready for custom development.

---

This process results in a modern, fast, and fully branded React app using Vite and Tailwind CSS, ready for further development.
