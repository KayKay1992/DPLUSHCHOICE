# Cart Slice Configuration - Shopping Cart State Management

## Overview
Implemented cart slice using Redux Toolkit for managing shopping cart state in the D'Plush Choice e-commerce application.

## Cart Slice Implementation

### File: `frontend/src/redux/cartRedux.js`

```javascript
import { createSlice } from "@reduxjs/toolkit";

const cartSlice = createSlice({
  name: "cart",
  initialState: {
    products: [],    // Array of cart items
    quantity: 0,     // Total number of items
    total: 0,        // Total cart value
    email: "",       // User email for cart association
  },
  reducers: {
    // Actions for cart management
  },
});

export const { addProduct, removeProduct, clearCart } = cartSlice.actions;
export default cartSlice.reducer;
```

## State Structure

### Initial State:
```javascript
{
    products: [],    // Array storing individual cart items
    quantity: 0,     // Total count of all items in cart
    total: 0,        // Sum of all item prices × quantities
    email: ""        // Associated user email
}
```

## Reducer Actions

### 1. `addProduct` Action
```javascript
addProduct: (state, action) => {
  state.products.push(action.payload);           // Add product to array
  state.quantity += 1;                           // Increment total quantity
  state.total += action.payload.price * action.payload.quantity;  // Add to total
  state.email = action.payload.email;            // Set user email
}
```
**Purpose**: Adds a product to the cart
**Payload Expected**: `{ id, price, quantity, email, ...other product data }`

### 2. `removeProduct` Action
```javascript
removeProduct: (state, action) => {
  const index = state.products.findIndex(
    (product) => product.id === action.payload.id  // Find product by ID
  );
  if (index !== -1) {
    state.quantity -= 1;                          // Decrement quantity
    state.total -= state.products[index].price * state.products[index].quantity;
    state.products.splice(index, 1);              // Remove from array
  }
}
```
**Purpose**: Removes a specific product from cart
**Payload Expected**: `{ id }`

### 3. `clearCart` Action
```javascript
clearCart: (state) => {
  state.products = [];    // Empty products array
  state.quantity = 0;     // Reset quantity counter
  state.total = 0;        // Reset total price
  state.email = "";       // Clear email association
}
```
**Purpose**: Completely empties the cart
**Payload Expected**: None (resets all cart state)

## Key Features

### **State Management:**
- **Immutable Updates**: Redux Toolkit handles immutability automatically
- **Centralized State**: Single source of truth for cart data
- **Real-time Calculations**: Automatic total and quantity updates

### **Data Flow:**
```
Component → Dispatch Action → Reducer → State Update → UI Re-render
```

### **Integration Points:**
- **User Authentication**: Email association for user-specific carts
- **Product Catalog**: Product data integration for cart items
- **Checkout Process**: Total and quantity data for order processing

## Usage Examples

### Adding to Cart:
```javascript
dispatch(addProduct({
  id: "prod123",
  name: "Hydrating Cleanser",
  price: 2500,
  quantity: 2,
  email: "user@example.com"
}));
```

### Removing from Cart:
```javascript
dispatch(removeProduct({ id: "prod123" }));
```

### Clearing Cart:
```javascript
dispatch(clearCart());
```
