Setting up pages for admin dashboad

## **Navigation and Page Links Implementation**

### **1. React Router Setup**
You implemented client-side routing using React Router v6 with the following key components:

```jsx
import { createBrowserRouter, RouterProvider, Outlet } from 'react-router-dom'
```

- **createBrowserRouter**: Creates a browser-based router for handling URL changes
- **RouterProvider**: Provides the router context to the entire application
- **Outlet**: Renders the matched child route component

### **2. Layout Component Structure**
You created a shared layout that includes the navigation menu and content area:

```jsx
const Layout = () => {
  return (
    <div className='flex'>
      <div>
        <Menu />
      </div>
      <div>
        <Outlet />
      </div>
    </div>
  )
}
```

This layout ensures the menu is always visible while the main content changes based on the current route.

### **3. Router Configuration**
Your router is configured with nested routes under a root layout:

```jsx
const router = createBrowserRouter([
  {
    path: "/",
    element: <Layout />,
    children: [
      { path: "/", element: <Home/> },
      { path: "/users", element: <Users/> },
      { path: "/products", element: <Products/> },
      { path: "/orders", element: <Orders/> },
      { path: "/banners", element: <Banners/> },
    ]
  }
])
```

### **4. Menu Component Navigation**
The Menu component uses React Router's Link component for navigation:

```jsx
import { Link } from "react-router-dom";
```

**Menu Items Structure:**
- **Dashboard Section**: Home, Profile
- **Management Section**: Users, Products, Orders  
- **Content Section**: Banners, Settings, Backups
- **Analytics Section**: Analytics, All Logs

Each menu item is defined with an icon, label, and path:

```jsx
const menuItems = [
  { icon: FaHome, label: "Home", path: "/" },
  { icon: FaUser, label: "Profile", path: "/profile" },
];
```

### **5. Link Component Implementation**
Each menu item uses the Link component with hover effects:

```jsx
<Link
  to={path}
  className="flex items-center space-x-4 px-5 py-4 rounded-2xl cursor-pointer transition-all duration-300 hover:scale-105 hover:bg-white/10"
>
  <Icon className="text-xl text-gray-400 group-hover:text-pink-400 transition-colors" />
  <span className="font-medium text-gray-300 group-hover:text-white">
    {label}
  </span>
</Link>
```

### **6. Page Components**
You created individual page components for each route:

- **Home.jsx**: Main dashboard with stats, transactions table, and chart
- **Users.jsx**: User management page
- **Products.jsx**: Product management page  
- **Orders.jsx**: Order management page
- **Banners.jsx**: Banner/content management page

### **7. Responsive Navigation**
The menu implements both desktop and mobile navigation:

- **Desktop**: Sticky sidebar that's always visible
- **Mobile**: Hamburger menu with slide-in overlay

### **8. Navigation Features**
- **Active State**: Visual feedback on hover with color transitions
- **Smooth Animations**: Scale and background effects on interaction
- **Icon Integration**: FontAwesome icons for each menu item
- **Accessibility**: Proper semantic HTML with Link components

### **9. Route Organization**
- **Nested Routing**: All routes are children of the root layout
- **Clean URLs**: Standard web URLs without hash routing
- **Component-Based**: Each route renders a specific React component

This implementation provides a professional, scalable navigation system that follows React Router best practices and maintains consistent styling with your glassmorphism theme.
