Creating Footer Component

Footer.jsx Implementation Details
-----------------------------------

1. Component Structure:
   - Footer.jsx is a functional React component that displays the site footer with modern styling.
   - It imports social media icons from react-icons (FaInstagram, FaTiktok, FaWhatsapp, FaFacebook).

2. Background and Layout:
   - Gradient background from gray-900 via purple-900 to pink-900 for a dark, luxurious feel.
   - Subtle pattern overlay using SVG for texture.
   - Responsive grid layout for upper part: logo/description, quick links, contact info.

3. Upper Part:
   - Logo with rounded ring and shadow.
   - Description text in gray-300.
   - Quick links with hover slide effect and color change.
   - Contact info with pink highlighted labels.

4. Lower Part:
   - Divider line in gray-700.
   - Copyright notice with dynamic year and heart emoji.
   - Social media icons with hover scale, translate, and color change, linked to provided URLs (Instagram, TikTok, WhatsApp, Facebook).

5. Features:
   - Eye-catching gradients and text effects using bg-clip-text.
   - Hover animations: scale, translate, color transitions.
   - Fully responsive with grid breakpoints.
   - Accessible with aria-labels on social links.

6. Styling:
   - Dark theme with pink/purple accents matching site branding.
   - Smooth transitions and transforms for interactivity.

This footer provides a luxurious, modern close to the site with all necessary links, contact info, and social media presence.
