ProductList Component Explanation
===============================

The ProductList component is a React functional component that serves as the main product catalog page for the D' Plush Choice e-commerce application. Here's a detailed breakdown of its structure and functionality:

## Component Overview
- **File Location**: frontend/src/pages/ProductList.jsx
- **Purpose**: Displays the complete product collection with filtering and sorting capabilities
- **Technology Stack**: React with hooks, Tailwind CSS for styling

## Key Features

### 1. State Management
- `selectedCategory`: Tracks the currently selected product category filter (default: "all")
- `sortBy`: Controls the sorting method for products (default: "featured")

### 2. Product Categories
The component includes 16 predefined categories based on the Category.jsx file:
- All Products (default)
- Perfumes
- Body Sprays
- Diffusers
- Scented Candles
- Necklace, Earrings, Rings, Wristwatches
- Anklets, Bracelets, Bangles
- Bags, Clutch Purse
- Jewelry Set, Other Accessories

### 3. Sorting Options
Six sorting methods are available:
- Featured (default)
- Price: Low to High
- Price: High to Low
- Newest First
- Highest Rated
- Most Popular

### 4. UI Structure

#### Header Section
- **Background**: Gradient from pink-50 to purple-50 with glassmorphism effects
- **Title**: Large gradient text "Our Product Collection"
- **Description**: Marketing text about the product collection

#### Filter & Sort Controls
- **Layout**: Responsive flex layout (column on mobile, row on desktop)
- **Styling**: Glassmorphism cards with backdrop blur and shadow effects
- **Category Filter**: Dropdown select with custom styling
- **Sort Options**: Dropdown select with matching design

#### Results Summary
- **Dynamic Text**: Shows current filter and sort selections
- **Position**: Centered below the controls

#### Products Display
- **Component**: Renders the Products component
- **Props**: Passes selectedCategory and sortBy to child component

## Styling Approach
- **Glassmorphism**: Extensive use of backdrop-blur, semi-transparent backgrounds
- **Gradients**: Pink to purple color scheme throughout
- **Responsive Design**: Mobile-first approach with breakpoint adjustments
- **Interactive Elements**: Focus states with ring effects and smooth transitions

## Event Handlers
- `handleCategoryChange`: Updates selectedCategory state when filter changes
- `handleSortChange`: Updates sortBy state when sort option changes

## Integration
- **Child Component**: Passes filter/sort state to Products component
- **Data Flow**: Props-based state sharing for filtering and sorting logic
- **UI Consistency**: Matches the overall application design system

## Current Status
- **UI Complete**: Fully styled with modern glassmorphism design
- **Functionality**: Currently UI-only (no backend integration as requested)
- **Responsive**: Works across all device sizes
- **Categories**: Aligned with luxury beauty and jewelry focus

This component provides a beautiful, user-friendly interface for browsing products while maintaining the luxury aesthetic of the D' Plush Choice brand.
